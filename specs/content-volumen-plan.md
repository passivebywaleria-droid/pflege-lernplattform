## 5. Content-Erstellungs-Strategie mit KI

### Reihenfolge fuer Content-Loop

**Phase 1 (MVP): CE 01 + CE 02** = 250 Std.
- 1.750 Texte + 4.500 Fragen + 375 Faelle + 7.500 Flashcards
- Geschaetzter Aufwand: ~200h mit KI-Unterstuetzung

**Phase 2: CE 03 + CE 05 (1./2. AD)** = 280 Std.
- Hoechste Pruefungsrelevanz (CE 05 = groesste CE!)
- ~280h mit KI

**Phase 3: CE 04 + CE 06 (1./2. AD)** = 140 Std.
- Praevention + Akutsituationen
- ~140h mit KI

**Phase 4: CE 07-11** = 490 Std.
- Restliche CEs der generalistischen Phase
- ~490h mit KI

**Phase 5: 3. AD Vertiefungen** = 380 Std.
- Drei Pfade (generalistisch, Kinder, Altenpflege)
- ~380h mit KI

### KI-gestuetzte Content-Generierung

Jede Lerneinheit (LE) wird als ein Ralph-Loop-Durchlauf generiert:

```
Fuer jede LE:
1. KI liest BIBB-Beschreibung + Kompetenzziele
2. KI generiert Lerntexte (paraphrasiert, nie kopiert!)
3. KI generiert MC-Fragen (mit Erklaerungen)
4. KI generiert Fallvignetten (kurz + lang)
5. KI generiert Flashcards
6. KI generiert vereinfachte Alternativtexte (fuer B1)
7. Dozentin prueft fachlich
8. Content wird in TypeScript-Dateien exportiert
```

### Output-Format pro LE

```typescript
// content/ce01/le-01-1.ts
export const LE_01_1 = {
  id: 'le-01-1',
  ceId: 'ce-01',
  title: { de: 'Ankommen in der Pflegeausbildung', ar: '...', tr: '...' },
  zeitrichtwert: 25, // Stunden
  texts: [...],      // 175 Lerntexte (Basis + vereinfacht)
  questions: [...],  // 450 Fragen (verschiedene Typen)
  cases: [...],      // 38 Fallvignetten
  flashcards: [...], // 750 Karteikarten
  glossary: [...],   // Fachbegriffe dieser LE
}
```
