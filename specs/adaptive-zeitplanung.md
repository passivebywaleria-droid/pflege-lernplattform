# Adaptive Zeitplanung — Evidenzbasierte Grundlagen

> Erstellt: 2026-03-28 | Quelle: Didaktik-Recherche (17 Studien)

## 1. Theoretisches Fundament

### Carroll's Model of School Learning (1963/1989)

```
Lerngrad = Zeit aufgewendet / Zeit benötigt
```

Schüler unterscheiden sich **nicht** in der Lernfähigkeit, sondern in der **Zeit zum Mastery**.
Drei Faktoren bestimmen "benötigte Zeit":
1. **Aptitude** — Lernanlage für das Thema
2. **Ability to understand instruction** — Sprachkompetenz (= unsere Achse 1)
3. **Quality of instruction** — Didaktik-Qualität (= unser System)

→ Bestätigt unseren Zwei-Achsen-Ansatz: Sprachkompetenz × Fachwissen = zwei getrennte Zeitbudgets.

### Bloom's Mastery Learning (1968)

- Effektstärke: **ES = 0.59** (Meta-Analyse) — moderat bis substanziell
- Bedingungen: Systematischer Unterricht + Hilfe bei Problemen + ausreichend Zeit + klares Mastery-Kriterium
- Zusatzeffekte: Verbesserte Selbstwirksamkeit, Motivation, Anwesenheit

### Competency-Based Education in Pflege (AACN 2021)

> "Zeit ist nicht das Kriterium — Mastery ist das Kriterium."

- AACN verlangt seit 2021 CBE für alle Nursing-Programme
- 92% der Pflegeschüler mit Adaptive Learning bevorzugen es (Wolters Kluwer)

## 2. Mastery-Schwellenwerte

### N-Consecutive Correct Responses (N-CCR)

| System | Schwellenwert | Kontext |
|--------|--------------|---------|
| ALEKS (unbekannt) | 5 richtige in Folge | Hoher Threshold |
| ALEKS (unsicher) | 3 richtige in Folge | Niedriger Threshold |
| ASSISTments | 3 richtige in Folge | Standard |
| Khan Academy | 4 Stufen: Attempted → Familiar → Proficient → Mastered | Gestuft |

### Accuracy-Schwellenwerte

- **90%**: Höhere Retention, aber mehr Lernzeit
- **80%**: Guter Kompromiss zwischen Effizienz und Retention
- **50%**: Unzureichend für Langzeit-Mastery

### Overlearning-Befund (kritisch!)

Mehr als 3-5 richtige in Folge zu verlangen erhöht die Lernzeit **ohne proportionalen Langzeit-Benefit**. Der Overlearning-Effekt verschwindet bei kognitiven Aufgaben nach **~38 Tagen**. Ausnahme: Motorische/prozedurale Skills.

→ **Spaced Repetition ist wichtiger als ein höherer Threshold.**

### Unsere Mastery-Entscheidung

```
Neues Thema (noch nie gesehen):
  → 3 richtige in Folge + Antwortzeit < 8s
  → Bei Fehler: Strategiewechsel, dann erneut prüfen

Schwaches Thema (hohe Fehlerrate):
  → 4-5 richtige in Folge
  → Spaced Repetition nach 1-2 Tagen zwingend

Bekanntes Thema (Wiederholung):
  → 1-2 richtige als Bestätigung
  → Dann direkt zu höherem Bloom-Level

Fast-Forward erlaubt wenn:
  → 3+ richtige in Folge
  → UND Antwortzeiten alle < 4s ("sicher")
  → UND kein Verwechslungsfehler
```

## 3. Antwortzeit als Signal

### 5-Stufen-Klassifikation (evidenzbasiert)

| Klasse | Zeitbereich | Signal | Konsequenz |
|--------|-------------|--------|------------|
| Blitz | < 1s | Reflexwissen oder geraten | Wenn richtig → echtes Wissen. Wenn falsch → geraten |
| Sicher | 1-4s | Abruf aus Langzeitgedächtnis | Mastery-Signal positiv |
| Normal | 4-10s | Verarbeitung + Abruf | Standard |
| Unsicher | 10-20s | Suche, Unsicherheit | Wissen fragil |
| Geraten | > 20s oder Timeout | Nicht gespeichert | Sofort Erklär-Step |

### Antwortzeit + Accuracy = kombiniertes Signal

- Konsistente Performance (schnell + richtig) = stabiler Wissenszustand
- Inkonsistente Performance (variabel) = schwacher Wissenszustand
- Schnell + falsch = geraten → Thema komplett neu erklären
- Langsam + richtig = fragiles Wissen → Spaced Repetition einplanen

Quelle: Response speed enhanced fine-grained knowledge tracing (2023)

## 4. Testing Effect — Die mächtigste Einzelerkenntnis

Karpicke & Roediger (2008, Science):

```
Gruppe A: 4× gelesen           → Retention nach 1 Woche: 36%
Gruppe B: 1× gelesen, 3× getestet → Retention nach 1 Woche: 80%
```

**Testen ist doppelt so effektiv wie Wiederlesen.**

→ Konsequenz: Auch kurze "Wissens-Pings" (3 Fragen zum alten Stoff) am Session-Start sind effektiver als Erklärtext-Wiederholungen.

## 5. Spaced Repetition — Konkrete Intervalle

### SM-2 Algorithmus (Wozniak 1987)

```
1. Wiederholung: nach 1 Tag
2. Wiederholung: nach 6 Tagen
Danach: Interval × Ease-Faktor (1.3–2.5)
```

### Optimale Lücke (Cepeda et al. 2008)

Spacing-Gap = **10-20% der gewünschten Retentionsperiode**:

| Behalten für | Wiederholung nach |
|-------------|-------------------|
| 1 Woche | 1 Tag |
| 1 Monat | 3-6 Tage |
| 1 Jahr | 5-7 Wochen |

### Optimale Recall-Wahrscheinlichkeit: 85-90%

Bjork's "Desirable Difficulties": Der optimale Wiederholungszeitpunkt ist wenn die Recall-Wahrscheinlichkeit bei **85-90%** liegt — schwer genug um das Gedächtnis zu stärken, nicht so schwer dass komplettes Relearning nötig ist.

### Wochen-Integration

```
Montag:     Neuer Stoff (LE 03)
Dienstag:   Neuer Stoff (LE 04) + 3 Fragen LE 03 (Tag 1)
Mittwoch:   Neuer Stoff + 3 Fragen LE 03 + 3 Fragen LE 04
Freitag:    Wiederholung: LE 03, LE 04, LE 05 (Tag 4-5)
Woche 2 Mo: 3 Fragen LE 03 (Tag 7) — Langzeit-Festigung
```

## 6. Session-Länge

### Evidenz

- "10-Min-Aufmerksamkeitsspanne" ist **nicht evidenzbasiert** (Physiological Review 2016)
- Aufmerksamkeit fluktuiert, fällt nicht linear ab
- Bei hohem Engagement: 50-Min-Sessions möglich
- Interaktive Elemente verlängern Engagement signifikant
- Video/Chunk Engagement-Optimum: **6-10 Minuten** (Guo et al. 2014)

### Optimale Session-Struktur (25-30 Min)

```
Min 1-3:    Spaced-Repetition-Ping (3 Fragen vom letzten Mal)
Min 4-12:   Neuer Stoff (Erklärstep + 4-6 Fragen, 2 Typen)
Min 13-20:  Vertiefung (komplexerer Fragetyp, Fallbezug)
Min 21-25:  Abschluss (Exit-Ticket, Zusammenfassung, XP)
```

### B1-Schüler: Kognitive Überlast-Faktor

Working Memory wird durch L2-Verarbeitung zusätzlich belastet:
- Kürzere Text-Blöcke (max 80 Wörter statt 150)
- Mehr Pausen zwischen Fragetypen
- Glossar on-demand (reduziert Working-Memory-Last)
- TTS-Funktion (entlastet visuellen Kanal)
- **+5 Min pro Session einplanen**

## 7. Zeitbudgets pro Schülertyp (CE 05, Ziel 40 UE)

```
Schüler A — Muttersprachler, Pflegeanfänger:
  Sprachzeit: minimal
  Fachzeit: voll (alle 16 LEs, volle Tiefe)
  Geschätzt: 35-40 UE ✓

Schüler B — B1, erfahrener Pflegehelfer:
  Sprachzeit: +20-30% pro Step
  Fachzeit: viele Steps Fast-Forward (zeigt schnell Mastery)
  Geschätzt: 35-40 UE ✓ (mehr Sprache, weniger Fach = gleiche Gesamtzeit)

Schüler C — B1, Pflegeanfänger:
  Sprachzeit: +20-30%
  Fachzeit: volle Tiefe
  Geschätzt: 45-55 UE ⚠ → braucht Puffer + priorisierte Themen

Schüler D — Muttersprachler, Pflegeerfahrung:
  Fast-Forward bei bekannten Themen
  Fokus: Lücken schließen, nicht komplette LEs
  Geschätzt: 20-25 UE → Challenges + Spaced Repetition füllen Rest
```

### Strategie für die 40-UE-Lücke

| Schülertyp | Kernzeit | Freigewordene Zeit füllen mit |
|-----------|----------|-------------------------------|
| Schnell (D) | 20-25 UE | Challenge-Steps (höherer Bloom), Prüfungssimulation, Cross-CE-Interleaving |
| Normal (A/B) | 35-40 UE | Passt. Spaced Repetition eingebaut |
| Langsam (C) | 45-55 UE | Priorisierung: nur prüfungsrelevante LEs voll, Rest gekürzt |
